# SoulFlame AR Business Card

Това е готов лек WebAR prototype, който разпознава черната страна на визитката чрез камерата и закрепя AR слой върху нея.

## Как да го стартираш

### Вариант 1: Vercel
1. Качи цялата папка в GitHub.
2. Import във Vercel.
3. Отвори HTTPS линка от телефон.
4. Натисни START CAMERA.
5. Насочи камерата към черната страна.

### Вариант 2: Локално
Камерата в браузър работи най-сигурно през localhost.

```bash
cd soulflame-ar-card
python -m http.server 3000
```

После отвори:

```txt
http://localhost:3000
```

## Важно

Този ZIP е без външни библиотеки. Разпознава голямата черна карта и лепи AR слой върху нея.

За следващата професионална версия ще заменим custom tracker-а с MindAR `.mind` target, което ще даде по-стабилно image tracking заключване.
